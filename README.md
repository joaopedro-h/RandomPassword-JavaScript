# 🔐 Gerador de Senhas

Aplicação web desenvolvida para gerar senhas seguras e personalizadas de forma rápida e prática.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Funcionalidades

- Geração de senhas aleatórias
- Definição de tamanho da senha
- Inclusão de:
  - Letras maiúsculas
  - Letras minúsculas
  - Números
  - Caracteres especiais
- Interface simples e intuitiva
- Atualização dinâmica via manipulação de DOM

## 🧠 Conceitos Aplicados

- Manipulação de DOM
- Eventos (`addEventListener`)
- Funções
- Estruturas condicionais
- Geração de números aleatórios (`Math.random`)
- Organização de lógica em JavaScript

## 🔒 Objetivo do Projeto

Projeto desenvolvido para praticar:

- Lógica de programação
- Segurança básica em geração de senhas
- Interatividade com JavaScript puro
- Estruturação de projetos front-end
